[문제2] Express , Diango, Rails 프레임워크에 대해서 조사한 후 
아래 순서에 따라 각각 정리하세요. (5점: ABCD평가)
 1) 정의
 2) 특징

Express
1) 정의
-> Node.js를 위한 빠르고 개방적인 웹 프레임워크
2) 특징
->https://jeong-pro.tistory.com/57  핵심 부분 확인
-> Node.js 프레임워크 중 커뮤니티가 가장 크다
-> 서버를 쉽게 실행/운영이 가능하다
-> 내장된 라우터로 코드 재사용이 가능

Django
1) 정의
-> 파이썬으로 만들어진 오픈소스 웹 프레임워크

2) 특징
-> 보안이 우수하고 유지보수가 편리한 웹사이트를 신속하게 개발할 수 있음
-> 비교적 쉬운 언어인 파이썬을 기반으로 두어 익히기 쉬움
-> 웹 애플리케이션이 제공되어야하는 사용자 인증, 관리 등이 기본으로 구현 되어있음

Rails
1) 정의
-> Ruby언어로 개발된 오픈소스 웹 프레임워크

2) 특징
-> M(model)V(view)C(controller) 아키텍쳐 구조
-> 제공하는 규칙을 지키면 필요한 개발 작업만 진행할 수 있어 빠르게 구현 가능
-> 같은 코드를 반복하지 않도록 다양한 자동화 옵션 제공
